# Claude Code Plugin Marketplace

A curated collection of Claude Code plugins by Anders Hagberg.

## Plugins

| Plugin | Description | Install |
|---|---|---|
| [long-running-harness](https://github.com/andershagberg/long-running-harness) | Structured harness for long-running AI agent tasks with planning, generator-evaluator feedback loops, progress tracking, and session resumption. | `claude plugin add andershagberg/long-running-harness` |

## Install a Plugin

```bash
claude plugin add andershagberg/<plugin-name>
```

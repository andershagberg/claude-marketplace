# Claude Code Plugin Marketplace

A curated collection of Claude Code plugins by Anders Hagberg.

## Add this marketplace

```bash
/plugin marketplace add andershagberg/claude-marketplace
```

## Install a plugin

```bash
/plugin install long-running-harness@claude-marketplace
```

Or browse available plugins in `/plugin > Discover`.

## Available Plugins

| Plugin | Description | Source |
|---|---|---|
| [long-running-harness](https://github.com/andershagberg/long-running-harness) | Structured harness for long-running AI agent tasks with planning, generator-evaluator feedback loops, progress tracking, and session resumption. | [GitHub](https://github.com/andershagberg/long-running-harness) |
